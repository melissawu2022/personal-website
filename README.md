# personal-website
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app), and was made with primarily React.js, javascript, CSS, and HTML.

## Current alternative to deploying with Github Pages
- Clone the repo
- cd into the directory of this project
- Assuming npm is installed, do npm start to deploy web page locally

## Potential fixes to some build errors in local environment
- If you get an error saying something like "'react-scripts' is not recognized as an internal or external command,", try npm install react-scripts --save
